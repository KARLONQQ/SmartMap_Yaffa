شرح تشغيل SmartMap_Yaffa (Ultra Royal Edition)

1) ارفع الملفات إلى GitHub Pages أو Netlify:
   - index.html
   - yaffa-logo.png
   - qr-poster.html

2) بعد النشر، انسخ رابط الموقع (مثل https://yourname.github.io/SmartMap_Yaffa/)
   ثم افتح qr-poster.html وعدّل السطر:
   const LANDING_URL = 'https://example.com/yaffa';

3) افتح qr-poster.html في المتصفح واطبع الـ QR.

ملاحظات:
- شاشة ترحيب سينمائية (8 ثوانٍ) مع موسيقى تعمل تلقائيًا ثم تتوقف قبل الانتقال.
- بعدها شاشة اختيار اللغة بتأثير دخول فاخر.
- ثم الشرح الكامل للمشروع باللغة المختارة.
- زر "العودة لاختيار اللغة" أعلى الشرح.

تم إعداد المشروع بواسطة الطالب: محمود أبوقاعود (Karlonqq)
