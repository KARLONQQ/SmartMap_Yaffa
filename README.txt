
شرح تشغيل SmartMap_Yaffa (Premium)

1) ارفع الملفات إلى GitHub Pages أو Netlify:
   - index.html
   - yaffa-logo.png
   - qr-poster.html

2) بعد النشر، خذ رابط الموقع (مثل https://yourname.github.io/SmartMap_Yaffa/)
   وافتح qr-poster.html وعدّل السطر:
   const LANDING_URL = 'https://example.com/yaffa';
   وضع الرابط الحقيقي بدلًا منه، ثم افتحه واطبع QR.

ملاحظات:
- شاشة الترحيب تعمل مع موسيقى سينمائية قصيرة ثم تتوقف تلقائيًا.
- بعد الترحيب تظهر شاشة اختيار اللغة، ثم الشرح الكامل باللغة المختارة.
- لا تُحفظ أي بيانات؛ كل شيء يعمل بدون تسجيل دخول.
